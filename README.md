# Publishing House Management System

A complete publishing house management system with public website and admin dashboard.

## Features

### Public Website

- Beautiful homepage with featured releases
- Weekly highlights schedule
- Complete catalogue browsing
- Responsive design

### Admin Dashboard

- 📊 Dashboard with statistics (revenue, orders, catalogue)
- 📚 Catalogue management (CRUD operations)
- 🛒 Order management with status tracking
- ⚙️ Settings management
- 🔐 Authentication (Email/Password & Phone/OTP)
- 📁 Image upload support

## Tech Stack

### Frontend

- React 18
- TypeScript
- Tailwind CSS
- React Router DOM
- Axios
- React Hook Form + Zod

### Backend

- Node.js
- Express.js
- PostgreSQL
- Session-based authentication
- Multer for file uploads

## Installation

### Prerequisites

- Node.js (v18 or higher)
- pnpm (v8 or higher)
- PostgreSQL (v14 or higher)

### Setup Steps

1. **Clone the repository**

```bash
git clone <your-repo-url>
cd PUBLISHING-HOUSE
```
